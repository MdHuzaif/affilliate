Best X399 Motherboard for Ryzen Threadripper 2 (2950x and 2990wx)

Published on : May 07,2021 by IstiakAhmed

Motherboards using X399 chipset mainly support TR4 socket AMD Ryzen Threadripper series CPU. Ryzen 2nd Generation CPU contains Zen+ 12nm processor architecture. AMD Ryzen Threadripper 2nd generation processors are developed with higher clock speed, increased power management, and a huge improvement in the number of cores.

Here is the overview of the Ryzen 2nd Generation Threadripper series:

-AMD Ryzen Threadripper 2920x which contains 12 cores, 24 threads, and CPU clock speed 3.5GHz to 4.3 GHz
-AMD Ryzen Threadripper 2950x which contains 16 cores, 32 threads, and CPU clock speed 3.5 GHz to 4.4 GHz
-AMD Ryzen Threadripper 2970WX which contains 24 cores, 48 threads, and CPU clock speed 3.0 GHz to 4.2 GHz
-AMD Ryzen Threadripper 2990WX which contains 32 cores, 64 threads, and CPU clock speed 3.0 GHz to 4.2GHz

Obviously, in this Threadripper 2nd Gen processor lineup the most highlighted one is 2990WX with massive 32 cores and 64 threads. There has been a noticeable upgrade in the 2990WX than the previous 1st Gen series that is capable of 4.2 GHz clock speed.

Here in the second generation Threadripper series we can see the “WX” suffix at the end of the name of two HEDT (high-end desktop) processors. AMD released this WX series as a consumer workstation processor. AMD released these two processors especially for content creators, innovators, 3D modeling, and video editors.

The AMD threadripper 2nd gen X series are mainly for streaming gamers who require more CPU capabilities for 4K live streaming while gaming. AMD has not changed the TR4 socket from the 1st gen to the 2nd gen Threadripper. So anyone with 1st gen Threadripper series processors can move to 2nd gen without changing the motherboard.

All X399 chipset-based motherboards support Ryzen Threadripper 2nd Gen processors. These motherboard CPU combinations provide tremendous overclocking performance and also power consumption. Old X399 motherboards will also support 2nd gen Threadripper CPUs without any issue. They are compatible with the latest BIOS.

As all X399 chipset motherboards are supported by Ryzen Threadripper 2nd gen processors but most of the X399 motherboards have run out of stock. So we carefully picked up 5 currently available X399 motherboards for your Ryzen Threadripper 2nd gen processor. All of the motherboards are award-winning and provide tremendous gaming, overclocking performance, high memory support, triple M.2 support, advanced LAN, audio, connectors, USB, and many other amazing features.


ASRock X399M Taichi

ASRock X399M Taichi is a micro ATX Ryzen Threadripper 2nd Generation motherboard. It takes the X399 massive spec list and tries to slap as much of it as possible onto the micro ATX form factor. This is not an easy task to do. Because X399 is enormous it has 64 PCIe 3.0, quad-channel memory, and up to 12 SATA ports. So we can easily understand why other manufacturers haven’t tried to do what ASRock has done with this motherboard.

This is the only Micro ATX motherboard available for Threadripper CPUs. Many people look for “best micro ATX motherboard for 2950X” and ASRock X399 Taichi is undoubtedly the best answer. This is also one of the cheapest currently available motherboards for Ryzen Treadripper 2950x and other 2nd gen processors. X339M is limited to two PCI express 3.0 two slot cards or three single-slot cards if you are water cooling. X399M is also able to run Quad channel memory. The max DDR4 capacity is cut in half than other X399 motherboards which are only 64 Gigabytes keep that in mind.

Also, there is one DIMM channel for each slot Total of 4 DIMM. Usual X399 motherboards provide two slots per channel. 8 SATA ports are found on the X399M motherboard. There is also a U.2 connector. ASRock managed to squeeze three M.2 slots on the board. One between the first and second PCI express slot. There are another two M.2 slots in the right section. The X399M rear I/O is sufficient 8 x 3.1 gen 1 port, 2 x 3.1 gen 2 port one being type C, Dual Gigabit LAN, 802.11 ac wifi, and a BIOS flashback. The VRM heatsink is pretty sufficient in size and the thin cutouts will help dissipate a lot of heat.

Aesthetically the ASRock X399M Taichi Motherboard looks really great, a nice neutral understated design. Despite being one of the cheaper X399 motherboards in the market it still looks and feels very premium. The RGB lighting is very limited. Only some subtle lighting underneath the chipset heatsink gear. ASRock X33M Taichi is one of the best motherboards for Threadripper 2950x and also 2970WX. This X399 micro ATX board is not recommended for Threadripper 2999WX processor.

ASRock Fatal1ty X399 Professional Gaming

ASRock Fatal1ty X399 professional gaming is a high-end threadripper motherboard. This is a gaming threadripper motherboard. The threadripper platform was not designed for gaming, it will do gaming pretty competently but it was designed for workstation-based configurations. This X399 motherboard has 60 amp chokes and 10-gigabit AQUANTIA Ethernet. ASRock Fatality X399 is an exceedingly nice motherboard. The layout of this motherboard is very interesting.

There is a single four-pin 12V connector on the top and on the other side of the RAM, there is CPU eight-pin power. In case of overclocking threadripper memory and also rise in memory if we route the CPU power connector over the RAM. As long as we clear the RAM by a decent amount there will be no issue of overclocking. Two CPU power connectors, a 4 pin power connector, and also a Chassis fan header connector. The heatsink has a heat pipe to carry heat to the back part of the motherboard. That's really good because the VRMs get quite hot while dumping 250 watts to power into the processor. Many people search asking about “best motherboard for threadripper 2990wx” and no doubt ASRock Fatal1ty X399 Professional Gaming motherboard is one of the best answers.

This X399 professional gaming motherboard supports 4 graphic cards. Each graphic card can consume up to 75 watts from the PCI slots. We might not be able to use some of the connections along the bottom of the motherboard if run graphics card because the 4th graphics card hang off the end of the motherboard so ASRock X399 professional gaming motherboard provided a right angle front panel connector for the audio. On the rear I/O, there is a BIOS flash button. We can use it to flash the UEFI on this motherboard.

The software package provided supports up to Sam Black Star Cinema 3. Creative Labs, Sound blaster, audio bundle, audio package all up to Cinema 3. This motherboard has 3 onboard M.2 ports. Two of which are 80 millimeters one is 110 millimeters. All the M.2 slots are directly into the CPU. This is the best possible situation as far as M.2 connectivity goes. There is not really a lot of motherboards to pick from for Treadripper and this is one of the more high-end boards we can get. There is not a lot of threadripper motherboards that actually do have built-in 10-gigabit ethernet which is an amazing feature to have when we have a processor as fast as threadripper.
